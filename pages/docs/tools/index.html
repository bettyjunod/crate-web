title: Tools Overview description: Management tools, utilities and data visualization for Crate author: Christian Haudum Crate plays well with others - and here are a couple of tools we connected Crate with or use ourselves: ## Management Utilities ### SQuirreL SQL An open-source, cross-platform Java SQL Client program for any JDBC compliant database. * [Download](http://squirrel-sql.sourceforge.net/) ### conZole A web-based SQL console for Crate written in Javascript. * [Github](https://github.com/megastef/crate-conZole) ### Crash Officially maintained shell for Crate written in Python. * [Reference](https://crate.io/docs/reference/crash/) * [Github](https://github.com/crate/crash) * [Download](https://pypi.python.org/pypi/crash) ### cr8 Utilities A collection of command line tools for Crate written in Python. * Blog: [A Collection of Utility Scripts for Working With Clusters](https://crate.io/a/cr8-utility-scripts/) * [Github](https://github.com/mfussenegger/cr8) * [Download](https://pypi.python.org/pypi/cr8) ### ctop A top/stat tool for Crate written in Python. * Blog: [A Tool for Crate Clusters](https://crate.io/a/ctop-top-for-crate-cluster/) * [Github](https://github.com/chaudum/crate-top) ## Data Visualization ### Tableau Desktop Tableau is a data visualization software that lets you see and understand data in minutes. **Connecting Tableau to a Crate cluster requires our ODBC driver, [contact us](info@crate.io) for more details.** * [Website](http://www.tableau.com/) ### Kibana Kibana is an open source data visualization platform that allows you to visualize and interact with your data. * Blog post: [How to use Kibana with Crate](http://techblog.bigdata-analyst.de/2015/06/25/how-to-use-kibana-4-with-crate-io/) * [Website](https://www.elastic.co/products/kibana) ### SQLPad SQLPad is a web-based SQL editor that can be run in your own private cloud and supports multiple databases. * [Website](http://rickbergfalk.github.io/sqlpad/) * [Github](https://github.com/rickbergfalk/sqlpad) ### Pentaho Pentaho data integration prepares and blends data delivering business analytics from any source. * [Website](http://community.pentaho.com) * [Github](https://github.com/pentaho/pentaho-kettle)
