title: Windows author: Chris W. date: 2015-11-30 # Windows ## Configure Windows To install Crate on Windows, you will need a [Java JDK installed](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html). Set the `JAVA_HOME` environment variable: ![Windows environment variables](win-envvar.png) Allow the ports Crate uses through the Windows Firewall: ![Windows Firewall ports](win-port.gif) ## Install Crate [Download the Crate Tarball](/install/#tarball), expand it and move to a convenient location. Start crate by running `bin/crate`.
