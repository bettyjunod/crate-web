Crate is a distributed database that allows you to create a flexible and scalable backend suited to a wide variety of purposes. Combined with Weave, developers working with a large number of containers can implement overlay networks tailored to their applications without touching infrastructure.

With Weave your applications can use a virtual network of containers as if they were all connected into the same infrastructure. Nodes on the Weave network can easily be exposed to the outside World or kept private to your internal network.

Benefits of using Crate with Weave

Multicast Support

True broadcast communication in the cloud, Crate containers are aware of each others with no need for manual configuration.

Simple Network Configuration

Easily expose or hide containers with a simple command and no need to juggle ports from container to host.

Security

If your network is behind a firewall or traffic needs encryption, Weave can handle it.
