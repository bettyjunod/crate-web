Crate is a distributed database that allows you to create a flexible and scalable backend suited to a wide variety of purposes. Combined with Mesosphere, developers can code for datacenter backends as if they were a local development machine.

Mesosphere abstracts computer resources such as CPU, Memory and Storage to enable you to treat your distributed Crate cluster as a single collection of resources through a common interface for resource management.

Benefits of using Crate with Mesosphere

  • Abstraction - No matter the size of your data center, all of its resources are treated as one pool.
  • Optimization - Target applications at the machines that can execute them effectively.
  • Automatic Healing - If a node fails, the node cluster adjusts accordingly and you wont experience any loss of data.
