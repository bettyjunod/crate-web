Use our one-step commands to try Crate in a sandboxed environment or install it on a Linux machine. (Make sure you have Java installed.)

Operating System

The Crate Team maintains the official YUM repositories for RPM-based releases. It is supported on Linux Distributions such as RedHat Enterprise Linux, CentOS, or Scientific Linux. We are supporting RHEL 6 and RHEL 7.

On our Ubuntu team page you can find our personal package archives (PPA). We maintain stable and testing channels.

More Information »

We provide Debian packages for the Wheezy distribution (7.x). They are hosted on our own repository.

More Information »

We publish buildfiles for Arch Linux AUR. Installing packages from the AUR is a simple process (Full instructions are available on the Arch Linux Wiki).

More Information »

For OS X use our tarball, or the simple try script. Be sure to install Java first.
