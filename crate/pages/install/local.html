Try in a local sandbox

Use our easy one-step command to launch Crate in a sandboxed environment:

# bash -c "$(curl -L try.crate.io)"

Import your own data or use the live Twitter stream from the 'Get Started' tab as sample data.

Hint: Try launching on two or more machines in the same network :)

Install on your local machine

To quickly install on your Linux machine, use our one-step command to install the correct repository for your package manager and download Crate.

# bash -c "$(curl -L install.crate.io)"

Crate will be running on port 4200. You can use the Web-UI (/admin) or connect with the command line utility.

Quick Install using Docker

The most convenient way to run Crate on your Linux host using Docker.

It will not only keep your host system clean but also allows you to scale your backend from a single machine to a multi-node cluster easily.

# docker pull crate
# docker run -p 4200:4200 crate

Then visit your host on port 4200 in the browser, e.g.
http://127.0.0.1:4200/admin/
