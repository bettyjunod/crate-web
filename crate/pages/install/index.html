    Quick Install using Docker

    The most convenient way to run Crate on your Linux host using Docker.

    It will not only keep your host system clean but also allows you to scale your backend from a single machine to a multi-node cluster easily.

    docker pull crate
    docker run -p 4200:4200 crate

    Then visit your host on port 4200 in the browser, e.g.
    http://127.0.0.1:4200/admin/

    Read more in-depth about installing Crate with Docker »

    Try in a Local Sandbox

    Use our easy one-step command to launch Crate in a sandboxed environment: Hint: Launch it on two or more machine in the same network :)

    bash -c "$(curl -L try.crate.io)"

    Insert your data or play around with the live Twitter stream as sample data.

    Requires Java 7 (update 55 or later) or Java 8 (update 20 or later) on OS X or Linux. Using earlier versions of Java may result in a loss of data.

    Install on Linux

    We support almost every major version of Linux in widespread use and their associated package managers. This is the best way to install Crate for production environments, especially if you're using a configuration management toolset such as Chef, Puppet, or Saltstack.
