Building PHP Applications with Crate

{% endblock %} {% block body %}

Crate is a distributed database for Docker that allows you to create a flexible and scalable backend suited to a wide variety of purposes.

Here you can find all the tools and resources you need to develop scalable PHP applications with the Crate backend.

Install & Develop

Our PHP packages are hosted on Packagist for easy inclusion in your projects. There is a PDO driver and a DBAL Driver that depends on the PDO driver. The code for both drivers is on GitHub.

  • PDO Driver on Packagist
  • DBAL Driver on Packagist
  • PDO Driver on GitHub
  • DBAL Driver on GitHub

Learn

Find documentation about using Crate with a PHP application and view sample code to get you started.

  • DBAL adapter instructions
  • Level7 Systems' blog post on using Crate with Symfony2
