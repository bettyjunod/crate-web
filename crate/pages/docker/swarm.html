What is it?

Docker Swarm is a tool that treats a group of individual hosts as one large virtual host. Swarm can combine hosts from different data centers, various operating systems and diverse hardware configurations. The project is still in beta (and may change) but already supports multiple scheduling strategies and integrates with other tools like Mesosphere.

Why use it?

Using Docker Swarm will enable you easily to deploy images across a Docker cluster from only a few commands to the master node. Swarm integrates with Docker Compose and Machine making a complete toolkit for Docker deployment, orchestration and management.
