Crate is a distributed database for Docker that allows you to create a flexible and scalable backend suited to a wide variety of purposes. By deploying and managing your Crate cluster using Docker, you benefit from scalable capacity and resilience, no matter how complex or large your application becomes.

Docker is an open platform for developers and sysadmins to build, ship, and run distributed applications. Containerized applications help to isolate dependencies, standardize the operating environment, and foster portability.

Portability

Data can be moved anywhere, no matter the dependencies or platform.

Get Started with Crate and Docker

Docker Machine

Machine is a Crate-compatible tool for easily provisioning a Docker host in numerous environments. With one command docker-machine will create new a host, install Docker and then enable you to use your local client to manage the instance.

Learn about Docker Machine

Docker Swarm

Swarm is a tool for native clustering of Docker containers. It can be used to create and scale a Crate cluster across multiple hosts. It supports several different platforms and integrates with other third party tools like Mesosphere and RancherOS.

Using Docker Swarm

Docker Compose

Compose is a Docker orchestration tool for defining and spinning up multiple container applications. Because Compose provides an easy way to define volumes, links, and configurations in a single file it is a great fit for deploying a Crate cluster.
