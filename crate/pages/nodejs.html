Install & Develop

Our Node.js packages are hosted on npm for easy inclusion in your projects. There are several drivers available, all written by community contributors.

  • node-crate is SQL Crate driver using the _sql endpoint REST API
  • crate-connect is a simple Crate driver
  • CrateJS is an extension to crate-connect that provides better methods for working with Crate
