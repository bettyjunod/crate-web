Install & Develop

Our Node.js packages are hosted on npm for easy inclusion in your projects. There are several drivers available, all written by community contributors.

  • Download node-crate, a SQL Crate driver using the _sql endpoint REST API
  • Download crate-connect, a simple Crate driver
  • Download CrateJS, an extension to crate-connect that provides better methods for working with Crate

Learn

Find documentation about using Crate with a node.js application and view sample code to get you started.
